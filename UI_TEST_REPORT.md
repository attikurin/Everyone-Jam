## 🆕 v1.1.0 で達成したユーザー要件

### 要件: 「一度閉じて開いた際に、違うIDになるか、新しいボードを開ける（新しいID発行）できるように」

✅ **完全実装＆動作確認済み**

#### 動作仕様
1. **`?board=` パラメータ無しでアクセス** → 自動でボードを開かず、**スタート画面（modal-start）** を表示
2. スタート画面では3つの選択肢:
   - 🆕 **新しいボードを作る** → `board_xxxxxxxxxxxx` 形式の新IDを発行（50回連続生成でも全てユニークなことを確認）
   - 🔗 **招待リンクから参加** → 貼り付けたURLから `board=` を抽出して該当ボードを開く
   - 🕒 **最近のボード** → localStorage に保存済みのボード一覧から選択（更新時刻順）
3. **ヘッダー** にも以下のボタンを追加:
   - `📄+` 新しいボード（確認ダイアログ後にID発行）
   - `📁` 最近のボードを開く
4. **既存ボード** は `?board=<ID>` のURLで直接開けば従来通りスキップしてアプリ起動

#### テスト確認項目（抜粋）
- ✅ P1-07 URLに `?board=` が無い場合 → スタート画面が出てKonva未起動
- ✅ P8-01 別タブで開いてもスタート画面表示（自動でボード開かない）
- ✅ P8-04 最近のボード一覧に既存ボードが表示される
- ✅ **P8-05 「新しいボードを作る」で前回と異なるIDが発行される（要件の核心）**
- ✅ P9-03 招待URL経由でアクセスすると既存ボードのタイトル「テストボード1」が正しく復元
